响应式图片

通过为图片添加 .ct-responsive类可以让图片支持响应式布局。其实质是为图片设置了 max-width: 100%;、 height: auto; 和 display: block;属性，从而让图片在其父元素中更好的缩放。

如果需要让使用了 .ct-responsive 类的图片水平居中，请使用 .ct-center-block 类

<img class="ct-responsive" alt="responsive image" />
<img class="ct-responsive ct-center-block" alt="responsive image" />
