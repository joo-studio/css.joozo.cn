We have included 150 Octicons Icons courtesy of Github. You can download them directly from the Github Octicons.

Usage

To be able to use these icons, you must include this line in the <head>portion of your HTML code. or download it in your projects.


  <link href="//cdn.bootcss.com/octicons/3.3.0/octicons.min.css" rel="stylesheet">

To use these icons, just place the name of the icon into the class of an span tag


  <span class="octicon octicon-arrow-right"></span>
