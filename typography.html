Headers

We provide some basic styling on header tags. In the example, you can see the the 6 header tags' different sizes.

To use headers on a body of text, simply just add the class ct-h1 to a tag


    <h1 class="ct-h1">Heading h1</h1>
    <h2 class="ct-h2">Heading h2</h2>
    <h3 class="ct-h3">Heading h3</h3>
    <h4 class="ct-h4">Heading h4</h4>
    <h5 class="ct-h5">Heading h5</h5>
    <h6 class="ct-h6">Heading h6</h6>

Flow Text

Toggle ct-text

One common flaw we've seen in many frameworks is a lack of support for truly responsive text. While elements on the page resize fluidly, text still resizes on a fixed basis. To ameliorate this problem, for text heavy pages, we've created a class that fluidly scales text size and line-height to optimize readability for the user. Line length stays between 45-80 characters and line height scales to be larger on smaller screens.

To see Flow Text in action, slowly resize your bct-rowser and watch the size of this text body change! Use the button above to toggle off/on flow-text to see the difference!


To use flow-text on a body of text, simply just add the class ct-text to a tag, see the code blow.


    <p class="ct-text">I am Flow Text</p>
