Usage

Here is a color palette based on the material design base colors. Each of these colors is defined with a base color class and an optional lighten or darken class.

Background Color

To apply a background color, just add the color darkness number with class to the element.

This is a card panel with a ct-bg-green class

    <div class="ct-card ct-bg-green">This is a card panel with a teal ct-bg-green class</div>

Text Color

To apply a text color, just replace bg to color like this: .ct-color-green

This is a card panel with dark green text
                        
    <div class="ct-card ct-pa-lg">
      <span class="ct-color-green">This is a card panel with dark blue text</span>
    </div>
