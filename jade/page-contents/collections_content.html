Badges can notify you that there are new or unread messages or notifications. Add the .--new class to the badge to give it the background.

Badge

Alan1 Alan4 Alan Alan14 

  <div class="collection">
    <a href="#!" class="__item">Alan<span class="__badge">1</span></a>
    <a href="#!" class="__item">Alan<span class="--new __badge">4</span></a>
    <a href="#!" class="__item">Alan</a>
    <a href="#!" class="__item">Alan<span class="__badge">14</span></a>
  </div>
