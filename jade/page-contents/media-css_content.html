Images

Responsive Images

To make images resize responsively to page width, you can add the class ct-responsive to your image tag. It will now have a max-width: 100% and height:auto.

To center images which use the ct-responsive class, use ct-center-block or ct-mx-autoIt will now have a margin-right: auto and margin-left: auto instead of ct-center. 


  <img class="ct-responsive" src="cool_pic.jpg">
  //Display In Center
  <img class="ct-responsive ct-center-block" src="cool_pic.jpg">
Circular Responsive images
This is a square image. Add the "ct-circle" class to it to make it appear circular.

To make images appear circular, simply add class="ct-circle" to them


  <div class="ct-md-8 ct-md-offset-2 ct-lg-6 ct-lg-offset-3">
    <div class="ct-card">
      <div class="ct-row ct-valign-wrapper">
        <div class="ct-sm-2">
          <img src="images/yuna.jpg" alt="" class="ct-circle ct-responsive"> <!-- notice the "circle" class -->
        </div>
        <div class="ct-sm-10">
          <span class="ct-color-black">
            This is a square image. Add the "ct-circle" class to it to make it appear circular.
          </span>
        </div>
      </div>
    </div>
  </div>

100% Images

To make images 100% to page width, you can add the class ct-img100 to your image tag. It will now have a width: 100% and height:auto.


  <img class="ct-img100" src="cool_pic.jpg">

Image shapes

Add classes to an <img>element to easily style images in any project.


  <img src="..." alt="..." class="ct-rounded">
  <img src="..." alt="..." class="ct-circle">
  <img src="..." alt="..." class="ct-thumbnail">

Videos

We provide a container for Embedded Videos that resizes them responsively.

Responsive Embeds

To make your embeds responsive, merely wrap them with a containing div which has the class .ct-video-container


  <div class="ct-video-container">
    <iframe width="853" height="480" src="http://player.youku.com/player.php/sid/XMTMxMzY5MzUzMg==/partnerid/6a9728f9675f9d36/v.swf" frameborder="0" allowfullscreen></iframe>
  </div>

Responsive Videos

To make your HTML5 Videos responsive just add the class ct-responsive to the video tag.


  <video class="ct-responsive" controls>
    <source src="movie.mp4" type="video/mp4">
  </video>
