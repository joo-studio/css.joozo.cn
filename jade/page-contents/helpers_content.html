Alignment

We have easy to use classes to help you align your content.

Vertical Align

You can easily vertically center things by adding the class .ct-valign-wrapper to the container holding the items you want to vertically align.

.ct-valign

    <div class="ct-valign-wrapper">
        <h5 class="__valign">This should be vertically aligned</h5>
    </div>

Text Align

These classes are for horizontally aligning content. We have .ct-left, .ct-right and .ct-center


This should be left aligned


This should be right aligned


This should be center aligned

    <div>
      <h5 class="ct-left">This should be left aligned</h5>
    </div>
    <div>
      <h5 class="ct-right">This should be right aligned</h5>
    </div>
    <div>
      <h5 class="ct-center">This should be center aligned</h5>
    </div>

Quick Floats

Quickly float things by adding the class .ct-fl or .ct-fr to the element. !important is used to avoid specificity issues.


    <div class="ct-fl">...</div>
    <div class="ct-fr">...</div>

Uppercased text.

Clearfix

Easily clear floatsby adding .ct-clearfix to the parent element. Utilizes the micro clearfix as popularized by Nicolas Gallagher. Can also be used as a mixin.


Usage

    <div class="ct-clearfix">...</div>

Hiding Content

We provide easy to use classes to hide content on specific screen sizes.

Class Name Screen Range
.ct-show Show for all Devices
.ct-hide .ct-hide-xs-up Hidden for all Devices
.ct-hide-xs-down Hidden for Extra small screen / phone down (0)
.ct-hide-sm-up Hidden for Small screen / phone Above (544px)
.ct-hide-sm-down Hidden for Small screen / phone Below (544px)
.ct-hide-md-up Hidden for Medium screen / tablet Above (768px)
.ct-hide-md-down Hidden for Medium screen / tablet Below (768px)
.ct-hide-lg-up Hidden for Large screen / desktop Above (992px)
.ct-hide-lg-down Hidden for Large screen / desktop Below (992px)
.ct-hide-xl-up Hidden for Extra large screen / wide desktop Above (1200px)
.ct-hide-xl-down Hidden for Extra large screen / wide desktop Below (1200px)
.ct-text-hide Text Hidden for All Screen

Usage

  <div class="ct-hide">Hidden for mobile only</div>

Formatting

These classes help format various content on your site.

Overflow

To truncate long lines of text in an ellipsis, add the class .ct-overflow to the tag which contains the text. See an example below of a header being truncated inside a card.

This is an extremely long title that will be truncated


  <h4 class="ct-overflow">This is an extremely long title that will be truncated</h4>
