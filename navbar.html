The navbar is fully contained by an HTML5 Nav tag. Inside a recommended container div, there are 2 main parts of the navbar. A logo or brand link, and the navigations links. You can align these links to the left or right.

Right Aligned Links

To right align your navbar links, just add a ct-fr class to your <ul> that contains them.

Logo
  • Sass
  • Components
  • JavaScript


  <nav class="ct-nav">
      <div class="__wrapper">
          <div class="ct-sm-12">
              <a href="#!" class="__logo">Logo</a>
              <ul class="ct-fr ct-hide-md-down">
                  <li><a href="#!">Sass</a></li>
                  <li><a href="#!">Components</a></li>
                  <li><a href="#!">JavaScript</a></li>
              </ul>
          </div>
      </div>
  </nav>

Centering the logo

The logo will center itself on medium and down screens, but if you want the logo to always be centered, add the --center class to your <a class="brand-logo">. You will have to make sure yourself that links do not overlap if you use this.

          <a href="#!" class="__logo --center">Logo</a>

              <li><a href="#!">Sass</a></li>

          </ul>
      </div>
  </nav>

Fixed Navbar

To make the navbar fixed, you have to add an outer wrapping div with the class ct-navbar-fixed. This will offset your other content while making your nav fixed. You can adjust the height of this outer div to change how much offset is on your content. 


  <div class="ct-navbar-fixed">
    <nav class="ct-nav">
      <div class="__wrapper">
        <a href="#!" class="__logo">Logo</a>
        <ul class="ct-fr ct-hide-md-down">
          <li><a href="#!">Sass</a></li>
          <li><a href="#!">Components</a></li>
        </ul>
      </div>
    </nav>
  </div>

Navbar Dropdown Menu

To add a navbar dropdown menu, add the ul dropdown structure into the page. Then, add an element to trigger the dropdown menu. Make sure to supply the id of the dropdown structure to the data-activates attribute of the dropdown trigger.

  • one
  • two
  • three
Logo
  • Sass
  • Components
  • Dropdown


  <!-- Dropdown Structure -->
  <ul id="dropdown1" class="ct-dropdown">
      <li><a href="#!">one</a></li>
      <li><a href="#!">two</a></li>
      <li class="ct-divider"></li>
      <li><a href="#!">three</a></li>
  </ul>
  <nav class="ct-nav">
      <div class="__wrapper">
          <a href="#!" class="__logo">Logo</a>
          <ul class="ct-fr ct-hide-md-down">
              <li><a href=".html">Sass</a></li>
              <li><a href=".html">Components</a></li>
              <!-- Dropdown Trigger -->
              <li><a class="ct-dropdown-button" href="#!" data-activates="dropdown1">Dropdown<i class="octicon octicon-triangle-down ct-ml-xs ct-fr"></i></a></li>
          </ul>
      </div>
  </nav>

To activate the dropdown menu, insert this line of code into your JavaScript file, within the $( document ).ready(function) block


$(".ct-dropdown-button").dropdown();
Trigger dropdown menu on click

By default, the dropdown menu is activated by hovering over the dropdown trigger. To activate the dropdown menu on click, pass { hover: false } into the above dropdown() function

Icon Links

Logo

You can add icons into links. For icon only links you don't need any additional class. Just pop the i tag in and it will work.


  <nav class="ct-nav">
      <div class="__wrapper">
          <a href="#!" class="__logo">Logo</a>
          <ul class="ct-fr ct-hide-md-down">
              <li><a href="#!"><i class="octicon octicon-mark-github"></i></a></li>
              <li><a href="#!"><i class="octicon octicon-graph"></i></a></li>
              <li><a href="#!"><i class="octicon octicon-globe"></i></a></li>
              <li><a href="#!"><i class="octicon octicon-search"></i></a></li>
          </ul>
      </div>
  </nav>

Logo

When your nav bar is resized, you will see that the links on the right turn into a hamburger icon . Take a look at the example below to get this functionality. Add the entire __collapse and ct-dropdown-button line to your nav.


  <nav class="ct-nav">
      <div class="__wrapper">
          <a href="#!" class="__logo">Logo</a>
          <a href="#" data-activates="mobile-demo" class="__collapse ct-ml-sm ct-dropdown-button"><i class="octicon octicon-three-bars"></i></a>
          <ul class="ct-fr ct-hide-md-down">
              <li><a href="#!">Sass</a></li>
              <li><a href="#!">Components</a></li>
              <li><a href="#!">Javascript</a></li>
              <li><a href="#!">Mobile</a></li>
          </ul>
          <ul class="side-nav" id="mobile-demo">
              <li><a href="#!">Sass</a></li>
              <li><a href="#!">Components</a></li>
              <li><a href="#!">Javascript</a></li>
              <li><a href="#!">Mobile</a></li>
          </ul>
      </div>
  </nav>

Initialization

After including the button-collapse line into your navbar, all you have to do now is place this code in your page's $( document ).ready(function(){}) code. This example below assumes you have not modified the classes in the above example. In the case that you have, just change the jQuery selector in the line below to match it.


    $(".ct-button-collapse").sideNav();
